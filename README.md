## Курсовой проект №2. "Парсинг и обработка вакансий с сайта HeadHunter"

## Цель проекта
- Разработать программу, которая будет получать информацию о вакансиях с платформы hh.ru в России, сохранять ее в файл и позволять удобно работать с ней: добавлять, фильтровать, удалять.
- Закрепить знания из пройденного модуля по ООП.

## Инструкция по установке

1. Клонируйте репозиторий:
```python
https://github.com/DoniyoRich/Course_Project_2_OOP.git
```

2. Установите зависимости:
```
poetry install
```

## Описание
В курсовом проекте реализованы следующие классы и модули:

- класс ***User***;
- класс ***VacancyOperationsABS***;
- класс ***Vacancy***;
- класс ***ParserABS***;
- класс ***HeadHunterAPI***;
- класс ***FileSaverABS***;
- класс ***ExcelSaver***;
- класс ***JSONSaver***;
- модуль ***main***;
- класс ***constants***;
- класс ***utils***.

Классы с суффиксами ABS - это абстрактные классы.
Краткое описание класса и его назначение его модулей приведено в докстрингах.
