## Описание

Курсовой проект представляет собой готовое приложение для работы с вакансиями с сайта hh.ru.
В проекте реализованы следующие классы и модули:

- класс ***User***. Реализует методы поискового запроса от Пользователя, фильтрации вакансий по ключевому слова, сортировки, удаления,
а также получения заданного Пользователем количества вакансий по размеру зарплаты. 
- класс ***VacancyOperationsABS***. Абстрактный класс для класса ***User***.
- класс ***Vacancy***. Класс для работы с вакансиями, создает объект ***Vacancy*** с некоторыми полями, выбранными из JSON ответа от API hh.ru.
В классе реализованы методы сравнения объектов по уровню максимальной зарплаты.
- класс ***ParserABS***. Абстрактный класс для класса ***HeadHunterAPI***.
- класс ***HeadHunterAPI***. Данный класс реализует непосредственный GET запрос на API сайта hh.ru, получает JSON ответ и сохраняет его на диске.
- класс ***FileSaverABS***. Абстрактный класс для классов ***ExcelSaver*** и ***JSONSaver***.
- класс ***ExcelSaver***. Класс реализует метод сохранения списка вакансий в файл формата .xlsx (Excel).
- класс ***JSONSaver***. Класс реализует метод сохранения списка вакансий в файл формата json.
- модуль ***main***. Основной модуль программы.
- модуль ***constants***. В модуле собраны константы, используемые в различных точках программы.
- модуль ***utils***. Модуль содержит небольшие вспомогательные утилиты, такие как отображение пользовательского меню, получение ответ от Пользователя
на различные запросы программы, операции конвертации и другие.

Точка входа в программу - модуль ***main***.
